# üî∑ **2. NODE CREATION WORKFLOW**

## **2.1 Create New Node**

Trigger: User presses ‚ÄúAdd Node‚Äù

Workflow:

1. System opens node creation form
2. User selects:

# üî∑ **3. NODE EDIT WORKFLOW**

# üî∑ **5. NODE POWER WORKFLOW**

## **5.1 Manual Power Entry**

For nodes like:

* OLT
* Splitters
* FAT
* Dome closures
* ATB

Workflow:

1. User selects node ‚Üí presses ‚ÄúPower Update‚Äù
2. Inputs:

   * Cumulative loss

4. System updates downstream nodes

## **5.2 Automatic Power Entry (Bluetooth Splicer/OTDR)**

Workflow:
